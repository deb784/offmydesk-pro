IMAGES FOLDER — Off My Desk Professionals
==========================================

Upload the following image files to this folder:

1. deb-hero.jpg
   → Your laptop/desk working photo (portrait orientation)
   → Used in: Hero section (main above-the-fold photo)
   → Recommended size: 920x1150px minimum

2. deb-about.jpg  
   → Your blazer/standing photo
   → Used in: About Deb section
   → Recommended size: 800x1000px minimum

3. logo-dark.png
   → Full "Off My Desk Professionals" logo (for dark/navy backgrounds)
   → Used in: Navigation bar and Footer
   → Recommended size: 200x80px, transparent background, white/lime text

4. logo-icon.png
   → Checkmark icon only (no wordmark)
   → Used in: Browser favicon
   → Recommended size: 64x64px or 128x128px

HOW TO ADD YOUR PHOTOS:
-----------------------
Option A — Via Netlify deploy:
  Drag all image files into this images/ folder before publishing.

Option B — Via Netlify Dashboard after deploy:
  Go to your site in Netlify > Deploys > drag new files in.

Option C — For ongoing updates:
  Connect your Netlify site to a GitHub repo and push image updates via Git.

NOTE: The website will show clean placeholder boxes if images are missing.
It will not break or look broken — it just won't show your photos until
you upload the actual files.
